Current features:
  Sound effect when eating an apple
  Game music
  Stats screen button
  Control screen button
  Multiple types of apples
    RED - Increases the length of the Snek that ate it
    YELLOW - Increases game speed
    RaInBOw - Winning apple (1/1000 spawn chance)
